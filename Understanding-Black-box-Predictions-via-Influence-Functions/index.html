    # 概要1

    • 影響関数を用いて，個々の学習データの有無や摂動が予測結果に与える影響を定式化
    • 効率的な計算手法の提案
    • 4つのユースケースの紹介
      • ネットワーク挙動の分析
      • ネットワークを混乱させる摂動の計算
      • ドメイン不適合の検知
      • ノイズラベルの検出
