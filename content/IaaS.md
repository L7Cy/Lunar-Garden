---
title: IaaS
tags: [  ]
aliases: [Infrastructure as a Service]
---
クラウドにあるネットワークやサーバなどのコンピューティングリソースを利用するサービス
経年劣化によるハードウェアの交換を自社で行う必要がない
クラウド事業者に委ねることで管理が必要ない
- 例
	-   Amazon Web Services(AWS)
	-   Microsoft Azure
	-   FUJITSU Cloud Service
	-   Google Cloud
	-   Alibaba Cloud
	-   IBM Cloud
	-   Oracle Cloud
	-   Red Hat
	-   NHN Cloud
